# 基于 OpenCV 的局部特征检测、描述与图像匹配

## 项目简介

本项目使用 OpenCV 实现了图像局部特征的检测、描述与匹配，具体包括：

- **ORB 特征检测**：提取模板图 (`box.png`) 和场景图 (`box_in_scene.png`) 的关键点与描述子，并可视化。
- **特征匹配**：使用暴力匹配器（Hamming距离）进行匹配，并按距离排序展示前50对匹配。
- **RANSAC 剔除误匹配**：基于匹配点估计单应矩阵，利用几何一致性剔除错误匹配，输出内点比例。
- **目标定位**：将模板图的四个角点通过单应矩阵投影到场景图中，绘制目标边框。
- **参数对比实验**：改变 ORB 的 `nfeatures` 参数（500, 1000, 2000），对比关键点数量、匹配数量、内点比例及定位成功率。
- **SIFT 对比实验（选做）**：使用 SIFT 算法进行相同流程，与 ORB 比较匹配质量与运行速度。

## 环境要求

- Python 3.8+
- OpenCV 4.x（建议 `opencv-python` 和 `opencv-contrib-python`）
- NumPy
- Matplotlib

安装依赖：

```bash
pip install opencv-python opencv-contrib-python numpy matplotlib
```

## 文件结构

```
.
├── box.png                  # 模板图像（待检测的物体）
├── box_in_scene.png         # 场景图像（包含物体的图像）
├── import cv2.py      # 主程序代码
├── README.md                # 本文件
└── 结果图片（由程序生成）:
    ├── box_keypoints_XXX.png
    ├── scene_keypoints_XXX.png
    └── 匹配图、定位图等（自动显示）
```

## 运行方法

1. 将 `box.png` 和 `box_in_scene.png` 放在与脚本相同的目录下。
2. 执行命令：
   ```bash
   python feature_matching.py
   ```
3. 程序将依次：
   - 显示 `box.png` 和 `box_in_scene.png` 的特征点可视化图。
   - 显示 ORB 初始匹配图（前50对匹配线）。
   - 显示 RANSAC 剔除错误匹配后的内点匹配图。
   - 显示场景图中目标定位的绿色边框。
   - 在控制台输出：
     - 关键点数量、描述子维度
     - 初始匹配总数
     - Homography 矩阵
     - RANSAC 内点数量及比例
   - 执行参数对比实验（500/1000/2000）并打印结果表格。
   - （可选）执行 SIFT 对比实验并显示定位结果。
